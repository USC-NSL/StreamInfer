Goal: Measure throughput and ITL at varying request rates for gptoss-120b
on AsyncMoE and SGLang EP16, using balanced gate profiles only.

CLUSTER & SYSTEM SETUP
=========================================================================
  Cluster:     8 nodes × 2 L40S = 16 GPUs (Sphere-16)
  Model:       gptoss_120b (36L, 128E, top-4, bf16)
  Gate:        balanced variants only
  AsyncMoE:    conda disag12 | cwd ~/DisagMoE     | MEM_FRAC=0.98
  SGLang:      conda sglang-fp | cwd ~/sglang-fake-prefill | MEM_FRAC=0.77

  Run AsyncMoE first, then SGLang. Never run simultaneously.
  Between each AsyncMoE run, restart Ray cluster.

  Output bounds:   min=256, max=512
  Max context len: 2048
  Retries:         AsyncMoE max 3, SGLang max 2
  No advanced_logging / sglang recorder
